# Joanne & Jin's Baby Shower Site

This project is a static Vite site for the baby shower invitation and RSVP page.

## Development

Install dependencies:

```bash
npm install
```

Start the local dev server:

```bash
npm run dev
```

## Build

Create a production build:

```bash
npm run build
```

## GitHub Pages

The project is configured to publish to:

`https://jandjbabyshower.github.io/`

Deployment is handled by the GitHub Actions workflow at
`.github/workflows/deploy-pages.yml`.

In the repository settings, set Pages to use `GitHub Actions` as the source.
